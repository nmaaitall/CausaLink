# CausaLink - Causal Inference Platform

Discover true causal relationships in your data using advanced statistical algorithms.

## Features
- Upload and explore datasets
- Statistical analysis and visualization
- Causal discovery using PC Algorithm and Hill Climb Search
- Compare multiple algorithms
- Export results as PNG/TXT

## How to Use
1. Upload your CSV dataset
2. Explore your data with statistical summaries
3. Run causal analysis to discover relationships
4. Compare different algorithms
5. Download results

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- pgmpy
